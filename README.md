Offline Rich Text Editor – Daftar
Yeh ek simple lekin powerful rich text editor hai jo bilkul offline kaam karta hai. Aap isme jo bhi likhte hain, wo browser ke localStorage mein save ho jata hai, taake page refresh karne ya browser band karne ke baad bhi data safe rahe.

Khasusiyat (Features)
    • Real-time Formatting: Text ko bold ya italic bana sakte hain.
    • Auto-Save: Har 60 second baad aur har text change par content apne aap save ho jata hai.
    • Word aur Character Count: Turant word aur character count dekhte rahein.
    • Import/Export: .txt file import kar sakte hain aur current text ko export kar sakte hain.
    • Print: Seedha browser se print karein.
    • Cleanup Bullets: Extra ya empty bullet points hata dein.
    • Keyboard Shortcuts: Ctrl+S (save), Ctrl+E (export), Ctrl+P (print), Ctrl+B (bold), Ctrl+I (italic).
    • Responsive Design: Mobile aur tablet par bhi achi tarah kaam karta hai.
    • Clear with Confirmation: Sab kuch clear karne se pehle confirmation modal aata hai.
    
Istemal ka Tariqa (How to Use)
    1. Editor mein likhein: Jaise hi aap type karna shuru karte hain, word count update hota hai.
    2. Formatting ke liye: Top par bold aur italic buttons use karein.
    3. Save karne ki fikar na karein: Har change apne aap save hota hai. "Last saved" time dikhta hai.
    4. Export: "Export" button dabane se aapka text .txt file mein download ho jata hai.
    5. Import: "Import" se koi bhi text file upload kar sakte hain.
    6. Print: "Print" button current content ko print karta hai.
    7. Cleanup: Agar bullet points bekar hain to "Cleanup" se saaf kar dein.
    8. Clear All: "Clear" button se modal open hoga, phir confirm karein.
    
Technologies Istemal Hui
    • HTML5
    • CSS3 (Flexbox, Gradient, Responsive)
    • Vanilla JavaScript (bina kisi library ke)
    • Browser LocalStorage
    • ContentEditable API
